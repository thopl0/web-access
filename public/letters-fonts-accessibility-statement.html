Accessibility statement for Letters Fonts

Letters Fonts is committed to making lettersfonts.com accessible, in accordance with WCAG 2.1 level AA.

Conformance status

The Web Content Accessibility Guidelines (WCAG) define requirements for designers and developers to improve accessibility for people with disabilities. This website is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Automated testing checked 23 of 50 success criteria; 27 require manual review and have not been independently audited. Automated testing reliably covers only part of WCAG, so the absence of detected issues is not a guarantee of full conformance.

Non-accessible content

The following WCAG success criteria are known to be unmet:

  • 1.4.3 Contrast (Minimum) (Level AA)
  • 3.1.1 Language of Page (Level A)
  • 4.1.2 Name, Role, Value (Level A)

Letters Fonts assessed the accessibility of this website by automated testing with webaccessibilitychecker.org. This statement reflects the most recent scan (June 20, 2026) and is updated automatically as the site is re-scanned.
